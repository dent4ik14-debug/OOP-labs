# Лабораторна робота №2: Обробка текстових даних (ООП)

**Виконав:** Солом'яний Денис
**Варіант:** NZK=20 (C3=2, C17=3)

## Функціонал
Програма аналізує введений текст і знаходить унікальні слова заданої довжини, які містяться лише в **питальних реченнях**.

1. **Тип текстової змінної:** `io.StringIO` (емуляція текстового потоку в пам'яті).
2. **Дія з текстом:** Пошук слів заданої довжини у питальних реченнях.
3. **Обробка винятків:** - Перевірка правильності номера залікової книжки.
   - Обробка порожнього тексту.
   - Валідація довжини шуканого слова (має бути цілим додатним числом).

## Логіка роботи (Regex)
Для розбору тексту використовуються регулярні вирази:
* `[^.!?]+\?` — пошук речень, що завершуються знаком питання.
* `\b\w+\b` — виділення окремих слів із речення без знаків пунктуації.

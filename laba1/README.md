# Лабораторна робота №1: Обчислення матриць (ООП)

**Виконав:** [Солом'яний Денис]
**Варіант:** NZK=20 (C5=0, C7=6, C11=9)

## Функціонал
1. Множення матриці $B$ (тип float64) на скаляр $a=5.5$.
2. Обчислення середнього значення кожного стовпця результуючої матриці $C$.
3. Обробка виняткових ситуацій (TypeError, ValueError).

## Тестування
Налаштовано GitHub Actions. Тести запускаються автоматично при кожному пуші.

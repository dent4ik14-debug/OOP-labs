# Лабораторна робота №3: Масиви об'єктів (ООП)

**Виконав:** Солом'яний Денис
**Варіант:** NZK=20 (C11=9)

## Функціонал
Програма демонструє роботу з масивами користувацьких об'єктів (класів), їх сортування за кількома параметрами та пошук ідентичного елемента.

1. **Клас `NPC`:** Містить 5 полів (ім'я, рівень, здоров'я, тип ставлення, локація).
2. **Перевизначення методів:**
   - `__eq__`: дозволяє коректно порівнювати два об'єкти за їх вмістом (полями), а не за адресою в пам'яті.
   - `__repr__`: забезпечує зручне та зрозуміле відображення списку персонажів у консолі.
3. **Сортування:** Виконано багатокритеріальне сортування за допомогою `lambda`:
   - Основний критерій: **Рівень (level)** — за зростанням.
   - Другорядний критерій: **Здоров'я (health)** — за спаданням.
4. **Пошук:** Реалізовано пошук об'єкта в масиві, що відповідає всім заданим полям.
